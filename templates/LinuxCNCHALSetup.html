This section lists the HAL configuration files loaded from the INI file for this system. HAL stands for Hardware Abstraction Layer, and it is the underpinnings of any linuxcnc system. The HAL, as configured in these files, controls how the system behaves; what inputs are connected to which outputs, and what devices and modules are loaded. For a detailed explaination, see the HAL User Manual.

The [HAL] section of the INI file determines which files are listed here. HAL files are listed in the [HAL] section, with the key names HALFILE and POSTGUI_HALFILE. See the Configure tab to edit the INI file.
